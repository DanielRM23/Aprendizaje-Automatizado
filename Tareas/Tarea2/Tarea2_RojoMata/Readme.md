### Requisitos

Para ejecutar los notebooks, se recomienda tener instalado:

- Python 3.x
- Jupyter Notebook

Librerías necesarias:

- `pandas`: Manipulación de datos con DataFrames.
- `numpy`: Operaciones numéricas y manejo de arrays.
- `matplotlib`: Visualización de datos 2D y 3D.
- `seaborn`: Gráficos estadísticos avanzados.
- `scikit-learn`: Preprocesamiento, modelos de clasificación y regresión, validación cruzada, métricas.
- `category_encoders`: Codificación de variables categóricas para modelos.

Puedes instalar todas con el siguiente comando:

```bash
pip install pandas numpy matplotlib seaborn scikit-learn category_encoders
```
