## Descripción

Este trabajo contiene los archivos y notebooks relacionados con la resolución de la **Tarea 4** del curso  
**Aprendizaje Automatizado**, semestre **2025-2**.

### Notebooks

- **Ejercicio1.ipynb**: Construcción del vocabulario visual, reducción de dimensionalidad y entrenamiento de clasificadores con BoVW.
- **Ejercicio2.ipynb**: Vectorización TF-IDF del dataset 20 Newsgroups y comparación de kernels en SVM.

### Data

Es necesario que los archivos Ejercicio1.ipynb y Ejercicio1.ipynb estén situados en la misma carpeta en donde se ecuentra la data (imagedb).

### Requisitos

Para ejecutar los notebooks, se recomienda tener instalado:

- Python 3.x
- Jupyter Notebook

Librerías necesarias:

- `numpy`: Operaciones numéricas y manejo de arrays.
- `scikit-learn`: Preprocesamiento, extracción de características, modelos de clasificación, validación cruzada y métricas.
- `matplotlib`: Visualización de resultados (opcional).
- `seaborn`: Gráficos de matrices de confusión (opcional).
- `joblib`: Para guardar modelos entrenados (opcional).

Puedes instalar todas con el siguiente comando:

```bash
pip install numpy scikit-learn matplotlib seaborn joblib
```
